Functions

timerpool:start (duration, callback)
Starts a new timer.

Parameters:

  • duration number Amount of time, in seconds, this timer will expire in.
  • callback func Function to run when the timer expires. The callback will receive the expiring timer as a parameter. If you need to associate extra information with the callback, consider providing a table with a __call metamethod for this parameter.

Returns:

    timer The newly started timer.
timerpool:clear ()
Clears the timerpool's internal collection of timers. This function is useful if you need a fresh timerpool, but you don't want to allocate a new one.
timerpool:size ()
Returns the number of timers in this timerpool.
timerpool:update (dt)
Updates the timerpool. This updates all the timers that this timerpool is managing. Timers that have expired will have their callbacks executed, and the timers themselves will be removed from the timerpool.

Parameters:

  • dt number Time elapsed in seconds.
