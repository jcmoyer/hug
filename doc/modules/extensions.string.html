split (s, p [, noempty=false])
Splits a string on a delimiter.

Parameters:

  • s string String to split.
  • p string Pattern to split on.
  • noempty bool If true, the resulting table will have any empty entries removed from it. If false or omitted, empty entries will be preserved.

Returns:

    table A table containing the pieces s was split into.
