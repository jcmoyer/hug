statemachine:pop ()
Pops the topmost gamestate from the statemachine stack. The state being popped will receive a call to onLeave with the underlying state as a parameter. Likewise, the state being transitioned to will receive a call to onEnter with the popped state as a parameter.

Returns:

    gamestate The gamestate popped from the stack.
