timer:evaluate (f)
Evaluates the given function using this timer's normalized remaining time. This is equivalent to f(1 - timer:remaining() / timer:duration()).

Parameters:

Returns:

timer:status ()
Determines whether or not this timer has finished running.

Returns:

    str The string 'active' if this timer should still be running. If the timer has been cancelled, then this function returns 'cancelled'. If the timer's duration has passed without this timer being cancelled, this function returns 'finished'.
timer:cancel ()
Flags this timer as cancelled. Cancelled timers will always have the status of 'cancelled'.
