new (r, g, b, a)
Creates a new color object.

Parameters:

  • r int or table Red value. Accepted values fall in the range of [0..255]. Alternatively, you can pass a table with 3 or more numerical components and omit the rest of the parameters.
  • g int or nil Green value. Accepted values fall in the range of [0..255].
  • b int or nil Blue value. Accepted values fall in the range of [0..255].
  • a int or nil Alpha value. Accepted values fall in the range of [0..255]. This component is not required.

Returns:

    color A new color object with the specified component values.
